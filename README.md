Krysta McAlister - Portfolio

A modern, responsive single-page portfolio built with React, Tailwind CSS, and Vite. This project showcases my work as a Full-Stack Developer and Branding Specialist.

1. Install Dependencies

Run the following command to install all necessary packages:

npm install


2. Add Your Assets

To make the portfolio fully personal, you need to add your specific images and resume file.

Create a folder named assets inside the src directory: src/assets/.

Place the following files into src/assets/:

3. Run Development Server

Start the local development server to view the site:

npm run dev


Open your browser to the link provided (usually http://localhost:5173).
